# ssh-connect
🐙 simple ssh manager
![demo](https://raw.githubusercontent.com/gko/ssh-connect/master/demo.gif)

## Installation
```bash
git clone --recursive https://github.com/gko/ssh-connect
# in your .bashrc
source ./ssh-connect/ssh-connect.sh
```
